Solarized Colors in Stylus and CSS

Nothing fancy, well OK a little fancy. Just needed something to add solarized to my Stylus and CSS style sheets and didn't see anything on npm or bower initially that had it — particularly the functions to allow use of transparency (opacity) mixed with the standardized solarized palette (ex: base03(0.8).

If you want to tweek or add SASS/SCSS functions for the same fork and pull request me.

Installation

For those that prefer to integrate node_modules/css-colors-solarized/{styles,scripts} into your grunt|gulp build process, (which you probably should):

npm install css-colors-solarized --save-dev

And if you insist ...

bower install css-colors-solarized
